⚙️ Configuración
Solo necesitas tu Google Client ID. Se hace una única vez.
1
Ve a console.cloud.google.com e inicia sesión.
2
Crea un proyecto → APIs y servicios → Biblioteca → activa Google Calendar API.
3
Credenciales → Crear credenciales → ID de cliente OAuth. Tipo: Aplicación web. En "Orígenes autorizados" pon la URL donde está esta app.
4
Copia el Client ID (termina en .apps.googleusercontent.com) y pégalo abajo.
Se guarda solo en este dispositivo/navegador. No se envía a ningún servidor.
